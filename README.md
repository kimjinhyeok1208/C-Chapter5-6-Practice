# 📘 C Language Practice – Chapter 5~6
이 저장소는 C 언어의 **수식과 연산자**, **조건문(if–else)** 을 학습하기 위한 연습 문제와 풀이 코드를 정리한 것입니다.
----------------------------------------------------------------------------------------------------------------
## 📝 Practice Problems
### 1️⃣ Arithmetic Expression Evaluation
여러 산술 연산자가 포함된 수식을 계산하는 프로그램을 작성한다.
**학습 포인트**
- 산술 연산자 (`+`, `-`, `*`, `/`, `%`)
- 연산자 우선순위
- 정수 나눗셈의 특징
----------------------------------------------------------------------------------------------------------------
### 2️⃣ Find Maximum Number and Check Even/Odd
세 개의 정수를 입력받아 **가장 큰 값**을 찾고,  
해당 값이 **짝수인지 홀수인지**를 판별하는 프로그램을 작성한다.
**학습 포인트**
- 중첩된 `if–else` 문
- 비교 연산자
- 나머지 연산자(`%`) 활용
----------------------------------------------------------------------------------------------------------------
### 3️⃣ Age-Based Ticket Price Calculator
사용자의 나이를 입력받아 아래 기준에 따라 요금을 계산한다.
| Age Range | Price |
|----------|-------|
| 0 ~ 7 | Free |
| 8 ~ 13 | 5000 |
| 14 ~ 19 | 8000 |
| 20 ~ 64 | 12000 |
| 65+ | 6000 |
**학습 포인트**
- 논리 연산자 (`&&`)
- 조건 범위 설정
- 예외 처리
